# Arduino Line Follower

## Descrizione
Progetto di un robot **line follower** sviluppato con **Arduino** come lavoro scolastico.  
Il sistema permette al robot di seguire autonomamente una linea nera su superficie chiara tramite sensori infrarossi e controllo dei motori.

⚠️ **Progetto attualmente in sviluppo.**
